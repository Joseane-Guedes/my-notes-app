# 🔧 Run Locally

Clone the project

```bash
  git clone git@github.com:Joseane-Guedes/my-notes-app.git
```

Go to the project directory

```bash
  cd my-notes-app.git
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```
